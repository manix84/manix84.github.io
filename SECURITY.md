# Security Policy 🔐

This project is a static personal portfolio site. It does not intentionally collect credentials, payment details, or private user data.

## Reporting a Security Issue

If you find a security issue affecting this site, please contact the repository owner directly using the contact details published on the site or GitHub profile.

Please include:

- 🔎 The affected URL or file.
- 🧾 A clear description of the issue.
- 🧪 Steps to reproduce, if applicable.
- 💻 Any relevant browser, device, or network details.

Do not publicly disclose security issues before they have been reviewed.
